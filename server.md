## Офис CR
### CR-DC
- На CR-DC разверните службу каталога на базе ALD Pro 3.X.X.
- Обеспечьте делегацию домена rea26.skills на CR-DC.
### CR-SRV
- Реализуйте центр сертификации предприятия на сервере CR-SRV.
- На CR-SRV реализуйте почтовый сервер.
- На сервере CR-SRV реализуйте централизованный сбор журналов.
- На CR-SRV разверните TFTP-сервер для создания резервных копий конфигураций маршрутизаторов.
- Напишите скрипт вызова утилиты snmpwalk на CR-SRV
- apt install snmpwalk 
### WIN-AD
- Пользователей необходимо импортировать с доменного контроллера на хосте WIN-AD. После переноса пользователей выключите его.
### CR-CLI
- Для проверки отправьте письмо с компьютера CR-CLI, от пользователя Lori пользователю Eva на компьютере BR-CLI.
- Подключите репозиторий и установите пакет на CR-CLI
